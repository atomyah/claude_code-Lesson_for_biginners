# ToDo リスト Web アプリ

ローカル専用のシンプルな To-Do リストアプリ。ブラウザで `index.html` を開くだけで動作する。

---

## 要件定義

### 背景・目的

個人の日常タスクをブラウザ上で手軽に管理するためのローカルアプリ。インターネット環境不要、サーバー不要で PC・スマートフォン両方から使える。

---

### 機能要件

#### タスク管理

| 機能 | 詳細 |
|------|------|
| タスク追加 | タイトル（必須）とメモ（任意）を入力して追加。`Enter` キーまたは「追加」ボタンで実行 |
| タスク削除 | 各タスクの削除ボタン（×）で即時削除。確認なし・取り消し不可 |
| 完了管理 | チェックボックスで完了/未完了を切り替え。完了タスクはグレーアウトしてリストに残す |
| 並び替え | ドラッグ&ドロップで自由に順序変更 |

#### 検索・絞り込み

| 機能 | 詳細 |
|------|------|
| フィルタータブ | 「全て」「未完了」「完了」の 3 タブでリストを切り替え |
| キーワード検索 | タイトル・メモを対象にリアルタイム絞り込み（大文字小文字区別なし） |

#### データ永続化

- LocalStorage にタスクデータを自動保存（キー: `todo-tasks`）
- ページリロード後もデータが保持される

---

### 非機能要件

| 項目 | 要件 |
|------|------|
| 動作環境 | Chrome / Safari / Firefox 最新版 |
| レスポンシブ | スマートフォン（375px〜）から PC（1280px+）まで対応 |
| オフライン動作 | ネットワーク接続不要（CDN を除く初回ロード後） |
| デプロイ | なし。ローカルのみで使用 |
| ビルドツール | 不要。`index.html` を直接ブラウザで開く |
| セキュリティ | ユーザー入力は `textContent` で挿入（XSS 対策済み） |

---

### データモデル

```json
{
  "id": "一意の文字列（時刻+乱数）",
  "title": "タスクタイトル（必須）",
  "memo": "メモ（空文字可）",
  "completed": false,
  "createdAt": "ISO8601 形式の日時文字列"
}
```

LocalStorage には上記オブジェクトの配列を JSON 文字列で保存する。

## 使い方

```

start index.html

# ブラウザで開く（Mac）
open index.html
```

## デザイン

- **テーマ**: クリーン / ミニマル
- **背景**: ライトグレー（`#f3f4f6`）
- **カード**: 白 + ドロップシャドウ
- **テキスト**: ダークグレー（`#1f2937`）
- **アクセント**: ブルー（`#3b82f6`）
- **完了タスク**: 打ち消し線 + グレーアウト（opacity 45%）
