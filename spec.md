# Twilight ToDo — 機能仕様書

## 1. プロジェクト概要

ローカル専用の To-Do リスト Web アプリ。ビルド不要。`index.html` をブラウザで直接開いて使う。

- **対象環境**: Chrome / Safari / Firefox（最新版）
- **データ保存**: LocalStorage（キー: `todo-tasks`）
- **デプロイ**: なし（ローカルのみ）

---

## 2. 機能要件

### 2.1 タスク追加

| 項目 | 詳細 |
|------|------|
| タイトル | 必須。空文字では追加不可 |
| メモ | 任意。複数行テキスト |
| 追加方法 | 入力フォームの「追加」ボタン または `Enter` キー |
| 追加位置 | リストの末尾に追加 |

### 2.2 タスク削除

- 各タスクカードに削除ボタン（×）を表示
- 削除時は確認ダイアログなしで即時削除
- 削除は取り消し不可

### 2.3 完了管理

- 各タスクにチェックボックスを表示
- チェックを入れると**完了状態**になる
  - タイトルに打ち消し線を表示
  - カードをグレーアウト（透明度を下げる）
- チェックを外すと未完了に戻る
- 完了タスクはリストに残り続ける

### 2.4 並び替え

- ドラッグ&ドロップで自由に順序変更
- ライブラリ: SortableJS v1.15.6（CDN）
- 並び替え後の順序を LocalStorage に保存

### 2.5 フィルター

タブ切り替えで表示を絞り込む。

| タブ | 表示対象 |
|------|---------|
| 全て | 未完了 + 完了 |
| 未完了 | チェックなしのタスクのみ |
| 完了 | チェック済みのタスクのみ |

### 2.6 キーワード検索

- 検索バーにテキストを入力すると、タイトル・メモを対象にリアルタイム絞り込み
- 大文字小文字を区別しない
- フィルタータブと組み合わせて動作する
- 検索バーが空になると全件表示に戻る

---

## 3. データモデル

```json
{
  "id": "uuid-string",
  "title": "タスクタイトル",
  "memo": "メモ（空文字可）",
  "completed": false,
  "createdAt": "ISO8601文字列",
  "order": 0
}
```

LocalStorage には上記オブジェクトの配列を JSON 文字列で保存する。

## 4. UI / UX 要件

### 4.1 レイアウト

| 要素 | 説明 |
|------|------|
| ヘッダー | アプリ名 + タスク件数バッジ |
| 入力フォーム | タイトル入力 + メモ入力（折りたたみ可） + 追加ボタン |
| 検索バー | フィルタータブの上に配置 |
| フィルタータブ | 全て / 未完了 / 完了 の 3 タブ |
| タスクリスト | タスクカードの縦並び |
| タスクカード | チェックボックス・タイトル・メモ（任意）・削除ボタン・ドラッグハンドル |

### 4.2 レスポンシブ

- スマートフォン（幅 375px〜）から PC（幅 1280px+）まで対応
- コンテンツ幅: 最大 600px、中央寄せ
- タッチ操作でドラッグ&ドロップが動作すること

### 4.3 デザイントーン

- **背景**: ホワイト / ライトグレー（`bg-gray-50`）
- **カード**: 白 + 薄いドロップシャドウ（`shadow-sm`）
- **テキスト**: ダークグレー（`text-gray-800`）
- **アクセント**: ブルー（`blue-500` / `blue-600`）
- **完了状態**: テキストに打ち消し線 + グレーアウト（`opacity-50`）

### 4.4 アニメーション

- タスク追加: フェードイン
- タスク削除: フェードアウト
- チェック切り替え: 即時（アニメーションなし）

## 5. 技術スタック

| 役割 | 技術 |
|------|------|
| マークアップ | HTML5 |
| スタイル | TailwindCSS Play CDN |
| スクリプト | Vanilla JavaScript（IIFE パターン） |
| ドラッグ&ドロップ | SortableJS v1.15.6（CDN） |
| データ永続化 | LocalStorage |

### 制約事項

- Tailwind の動的クラス（JS テンプレートリテラル内のクラス名）は使用禁止。動的スタイルはインラインスタイル (`element.style.xxx`) で適用する
- ビルドツール・サーバー不要
- 全コードを `index.html` 1 ファイルに集約
